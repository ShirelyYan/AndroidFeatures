SUMMARY
MobileSpy is a commercial monitoring tool that logs certain information on the device it was installed on.
TECHNICAL DETAILS
MobileSpy is a commercial monitoring tool that is available from mobile-spy.com. It logs the following information on the device, and forwards them to a remote server:
• Calendar events
• Contacts
• Emails
• GPS locations
• Memos
• Phone calls
• Photos and videos
• SMS messages
• Tasks
• Visited URLs
The later variants is equipped with a remote wipe feature, which enables users to remove data from the device's SD card calendar, call log, contact, image, and URL bookmark. This feature can be activated through an SMS message containing specific text content.
